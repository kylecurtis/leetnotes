## Algorithm List (WIP)

### **1. Sorting Algorithms**

These are fundamental for many problems:

- **Bubble Sort**
- **Selection Sort**
- **Insertion Sort**
- **Merge Sort**
- **Quick Sort**
- **Heap Sort**
- **Counting Sort**
- **Radix Sort**
- **Bucket Sort**
- **Shell Sort**
- **Timsort** (used in Python and many real-world applications, a great optional addition)

### **2. Searching Algorithms**

Basic and more advanced search algorithms:

- **Linear Search**
- **Binary Search** (Iterative and Recursive)
- **Ternary Search**
- **Exponential Search**
- **Interpolation Search**
- **Jump Search**

### **3. Divide and Conquer**

Some important divide-and-conquer techniques beyond sorting and searching:

- **Closest Pair of Points**
- **Karatsuba Algorithm for fast multiplication**
- **Strassen's Matrix Multiplication**

### **4. Dynamic Programming**

Dynamic programming (DP) is crucial for solving optimization problems:

- **0/1 Knapsack Problem**
- **Longest Common Subsequence**
- **Longest Increasing Subsequence**
- **Edit Distance**
- **Subset Sum Problem**
- **Coin Change Problem**
- **Matrix Chain Multiplication**
- **Rod Cutting Problem**
- **Fibonacci Numbers (Tabulation and Memoization)**
- **Egg Dropping Problem**
- **Maximum Subarray (Kadane’s Algorithm)**
- **Minimum Path Sum (Grid-based DP)**
- **Longest Palindromic Subsequence**
- **Catalan Numbers**
- **DP on Trees** (Tree-DP, for problems related to trees like Diameter of a tree, etc.)
- **Bitmask DP** (Used in problems like Traveling Salesman Problem, etc.)
- **Knuth's Optimization** (For certain classes of DP problems)

### **5. Greedy Algorithms**

Greedy algorithms are often used when making locally optimal choices leads to a globally optimal solution:

- **Huffman Coding**
- **Activity Selection Problem**
- **Fractional Knapsack Problem**
- **Kruskal’s Algorithm (Minimum Spanning Tree)**
- **Prim’s Algorithm (Minimum Spanning Tree)**
- **Dijkstra’s Algorithm (Shortest Path)**
- **Job Sequencing Problem**
- **Coin Change Problem (Greedy Approach)**
- **Interval Scheduling**

### **6. Graph Algorithms**

Graph theory is central to competitive programming:

- **Breadth-First Search (BFS)**
- **Depth-First Search (DFS)**
- **Dijkstra’s Algorithm**
- **Bellman-Ford Algorithm**
- **Floyd-Warshall Algorithm**
- **Kruskal’s Algorithm**
- **Prim’s Algorithm**
- **Tarjan’s Algorithm (Strongly Connected Components)**
- **Kosaraju’s Algorithm (Strongly Connected Components)**
- **Topological Sorting (Kahn's Algorithm, DFS-based)**
- **Union-Find/Disjoint Set Union (with Path Compression and Union by Rank)**
- **Bridges and Articulation Points**
- **Bipartite Check**
- **Eulerian Path and Circuit**
- **Hamiltonian Path**
- **Max Flow Algorithms (Ford-Fulkerson, Edmonds-Karp, Dinic's Algorithm)**

### **7. Advanced Graph Algorithms**

For more complex graph problems:

- **2-SAT Problem**
- **Hungarian Algorithm (for Maximum Bipartite Matching)**
- **Hopcroft-Karp Algorithm (Maximum Matching in Bipartite Graphs)**
- **Tree Decomposition**
- **Heavy-Light Decomposition (HLD)**
- **Centroid Decomposition**

### **8. Number Theory**

Many problems involve number theory:

- **Greatest Common Divisor (GCD) using Euclid’s Algorithm**
- **Extended Euclidean Algorithm**
- **Modular Exponentiation**
- **Sieve of Eratosthenes**
- **Segmented Sieve**
- **Primality Tests (Fermat’s Test, Miller-Rabin Test)**
- **Prime Factorization using Pollard's Rho Algorithm**
- **Chinese Remainder Theorem**
- **Euler’s Totient Function**
- **Modular Inverse**
- **Exponentiation by Squaring**
- **Fibonacci Numbers (Matrix Exponentiation)**
- **Linear Diophantine Equations**
- **Integer Factorization**

### **9. Combinatorics**

Useful in problems dealing with combinations and permutations:

- **Binomial Coefficient (with Pascal’s Triangle)**
- **Combination Generation**
- **Permutation Generation**
- **Catalan Numbers**
- **Burnside's Lemma**
- **Inclusion-Exclusion Principle**
- **Floyd’s Tortoise and Hare (Cycle Detection in Sequences)**

### **10. String Algorithms**

Many problems involve efficient string manipulation:

- **Naive String Matching**
- **Knuth-Morris-Pratt (KMP) Algorithm**
- **Rabin-Karp Algorithm**
- **Z-Algorithm**
- **Aho-Corasick Algorithm (for multi-pattern search)**
- **Suffix Array**
- **Suffix Tree**
- **Trie Data Structure**
- **Longest Common Prefix (LCP) Array**
- **Manacher’s Algorithm (for Longest Palindromic Substring)**
- **Rolling Hash (with Polynomial Hashing)**
- **Minimum Lexicographic Rotation**

### **11. Geometry Algorithms**

Geometry-related problems often appear in contests:

- **Convex Hull (Graham Scan, Jarvis March)**
- **Line Intersection**
- **Point in Polygon (Ray Casting, Winding Number)**
- **Closest Pair of Points**
- **Rotating Calipers (for various geometric problems)**
- **Polygon Area**
- **Circle-Line Intersection**
- **Voronoi Diagram**
- **Delaunay Triangulation**

### **12. Data Structures**

Data structures are essential for efficient algorithms:

- **Array**
- **Linked List**
- **Stack**
- **Queue**
- **Deque**
- **Binary Search Tree**
